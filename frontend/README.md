# Grior - Frontend

## Description

Frontend of our 2022 web project named Grior.

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)

### Installing

Install dependencies

```
npm i
```

## Getting Started

Run the app

```
npm start
```
